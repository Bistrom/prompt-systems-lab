# WordPress pro bono

Projekt pro bono platformy WordPress dla inicjatywy Warsaw Tea Foundation.

## Status

Projekt jest na etapie startowym. Na razie utworzono tylko czysty katalog projektowy w repozytorium.

## Cel

Docelowo projekt ma służyć do rozwijania koncepcji i implementacji platformy WordPress, oddzielonej od narzędzi automatyzacyjnych.

## Ważne rozdzielenie

Automatyzacja daily logów znajduje się osobno w:

`05_AUTOMATION/local-daily-pipeline`

Ten katalog jest miejscem dla projektu produktowego WordPress pro bono, a nie dla pipeline'u daily logów.
